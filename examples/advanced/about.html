About This Plugin

The vite-plugin-html-srcset automatically generates responsive images during your build process, making it easy to deliver optimized images to users across all devices.

This plugin supports multiple image formats including AVIF, WebP, PNG, and JPEG, and can generate images at various sizes to ensure optimal loading performance.

Technical Features

High Performance

Optimized image delivery reduces bandwidth usage and improves load times.

Multiple Formats

Supports modern formats like AVIF and WebP with intelligent fallbacks.

Fully Responsive

Automatic generation of multiple image sizes for different screen densities.
