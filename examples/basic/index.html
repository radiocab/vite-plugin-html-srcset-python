How It Works

  1. Add ?srcset to any image src attribute
  2. Run pnpm run build
  3. The plugin automatically generates multiple image sizes and formats
  4. The HTML is updated with proper srcset attributes

Check the network tab in your browser's developer tools to see the optimized images being served!
